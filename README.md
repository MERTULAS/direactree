# Direactree

React UI Component for directory structure.

## Installation

```bash
npm install direactree
# or
yarn add direactree
```

## Usage

```jsx
import { MyComponent } from 'direactree';

function App() {
  return (
    <div>
      <MyComponent />
    </div>
  );
}
```

## Development

### Installing Dependencies

```bash
npm install
# or
yarn
```

### Development with Storybook

```bash
npm run storybook
# or
yarn storybook
```

### Building

```bash
npm run build
# or
yarn build
```
